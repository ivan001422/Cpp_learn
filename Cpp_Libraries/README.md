# 📚 Основные библиотеки C++

Краткое описание ключевых библиотек стандартной библиотеки C++.

---

## 📐 `<cmath>` — математические вычисления

Библиотека для выполнения математических операций.

```cpp
#include <cmath>
```

**Основные функции:**
- `sin()`, `cos()`, `tan()` — тригонометрические функции
- `sqrt()` — квадратный корень
- `pow()` — возведение в степень
- `abs()` — абсолютное значение
- `round()`, `ceil()`, `floor()` — округление
- `exp()`, `log()` — экспонента и логарифмы

---

## 📝 `<cstring>` — работа со строками C-style

Функции для работы с массивами символов (нуль-терминированные строки).

```cpp
#include <cstring>
```

**Основные функции:**
- `strlen()` — длина строки
- `strcpy()`, `strncpy()` — копирование строк
- `strcat()`, `strncat()` — конкатенация строк
- `strcmp()`, `strncmp()` — сравнение строк
- `strchr()`, `strstr()` — поиск символа/подстроки
- `memset()` — заполнение памяти
- `memcpy()`, `memmove()` — копирование блоков памяти

---

## 🎲 `<cstdlib>` — общие утилиты

Библиотека с набором вспомогательных функций.

```cpp
#include <cstdlib>
```

**Основные возможности:**
- `rand()`, `srand()` — генерация случайных чисел
- `malloc()`, `free()` — динамическое выделение памяти (C-style)
- `system()` — выполнение системных команд
- `exit()` — аварийное завершение программы
- `atoi()`, `atof()` — преобразование строк в числа
- `abs()` — абсолютное значение (целые числа)
- `qsort()` — быстрая сортировка

---

## 📤 `<iostream>` — ввод и вывод

Основная библиотека для потокового ввода/вывода данных.

```cpp
#include <iostream>
```

**Основные возможности:**

| Объект | Назначение | Пример |
|--------|------------|--------|
| `std::cin` | Ввод с клавиатуры | `int x; std::cin >> x;` |
| `std::cout` | Вывод на экран | `std::cout << "Hello";` |
| `std::cerr` | Вывод ошибок | `std::cerr << "Error!";` |
| `std::clog` | Логирование | `std::clog << "Log message";` |

---

## 📦 `<vector>` — динамический массив

Один из основных контейнеров Standard Template Library (STL).

```cpp
#include <vector>
```

**Основные операции:**
```cpp
std::vector<int> v;        // создание пустого вектора
v.push_back(10);           // добавление в конец
v.pop_back();              // удаление последнего
v.size();                  // текущий размер
v.empty();                 // проверка на пустоту
v[0];                      // доступ по индексу
v.at(0);                   // доступ с проверкой границ
v.begin(), v.end();        // итераторы
v.insert(v.begin(), 5);    // вставка
v.erase(v.begin());        // удаление
v.clear();                 // очистка всех элементов
```

**Преимущества:**
- Динамическое изменение размера
- Быстрый доступ по индексу — O(1)
- Эффективная итерация
- Безопасное управление памятью
- Совместимость с алгоритмами STL
