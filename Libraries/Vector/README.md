## Векторы C++

В C++ обычные массивы имеют фиксированный размер, задаваемый на этапе компиляции. Их нельзя изменить после создания:

```cpp
int array[n]; // n должно быть константой времени компиляции
```

Для работы с массивами переменного размера в стандартной библиотеке предусмотрен **вектор** (`std::vector`) — динамический массив, который может расти и сжиматься во время выполнения.

---

## Вектор: объявление и инициализация

Вектор — это шаблонный класс, хранящий элементы одного типа. Основные способы создания:

```cpp
#include <vector>
using namespace std;

vector<int> v1;                // пустой вектор
vector<int> v2 = {1, 2, 3, 4}; // инициализация списком
vector<int> v3(5);             // 5 элементов, все равны 0
vector<int> v4(5, 2);          // 5 элементов, все равны 2
vector<int> v5(v2);            // копирование (глубокое)
```

Доступ к элементам осуществляется по индексу, как в обычном массиве:

```cpp
cout << v2[0]; // 1
```

## Базовые операции с вектором

| Операция                | Описание                                   | Пример                      |
|-------------------------|--------------------------------------------|-----------------------------|
| `push_back(value)`      | Добавляет элемент в конец                  | `v.push_back(10);`          |
| `pop_back()`            | Удаляет последний элемент                  | `v.pop_back();`             |
| `insert(pos, value)`    | Вставляет элемент по итератору             | `v.insert(v.begin(), 0);`   |
| `erase(pos)` / `erase(first, last)` | Удаляет элемент или диапазон      | `v.erase(v.begin()+2);`     |
| `clear()`               | Удаляет все элементы                       | `v.clear();`                |
| `size()`                | Возвращает количество элементов            | `v.size()`                  |
| `empty()`               | Проверяет, пуст ли вектор                  | `if (v.empty()) ...`        |

Пример:

```cpp
vector<int> v = {1, 2, 3};
v.push_back(4);       // {1,2,3,4}
v.pop_back();         // {1,2,3}
v.insert(v.begin(), 0); // {0,1,2,3}
v.erase(v.begin()+1); // {0,2,3}
cout << v.size();     // 3
v.clear();            // {}
```

## Доступ к элементам

Существует несколько способов получить доступ к элементам:

- **`operator[]`** — прямой доступ по индексу (без проверки границ).
- **`at(index)`** — доступ с проверкой границ (бросает исключение при выходе за пределы).
- **`front()`** — ссылка на первый элемент.
- **`back()`** — ссылка на последний элемент.
- **`data()`** — указатель на внутренний массив (для совместимости с C-функциями).

```cpp
vector<int> v = {10, 20, 30};
cout << v[0] << endl;          // 10
cout << v.front() << endl;     // 10
cout << v.back() << endl;      // 30
cout << *v.data() << endl;     // 10 (указатель на первый элемент)
```

## Итераторы: основа работы с контейнерами

Итераторы — это обобщённые указатели, позволяющие перебирать элементы контейнера.

- `begin()` — итератор на **первый** элемент.
- `end()` — итератор на **позицию за последним** элементом (не разыменовывается!).

```cpp
vector<int> v = {5, 2, 3, 1, 2, 5, 7, 1};
auto it = v.begin();
cout << *it; // 5
```

> **Важно:** `end()` указывает за пределы контейнера, поэтому для доступа к последнему элементу используют `back()` или `rbegin()`.

---

## Обход вектора с помощью итераторов

Три основных способа обойти вектор:

```cpp
// 1. По индексу
for (size_t i = 0; i < v.size(); ++i)
    cout << v[i] << " ";

// 2. Range-based for (C++11)
for (int x : v)
    cout << x << " ";

// 3. С помощью итераторов
for (auto it = v.begin(); it != v.end(); ++it)
    cout << *it << " ";
```

## Стандартные алгоритмы для векторов

Заголовочный файл `<algorithm>` предоставляет множество функций, работающих с диапазонами итераторов.

| Функция                | Описание                                                      |
|------------------------|---------------------------------------------------------------|
| `sort(first, last)`    | Сортирует диапазон по возрастанию                             |
| `reverse(first, last)` | Меняет порядок элементов на противоположный                  |
| `random_shuffle(first, last)` | Перемешивает элементы случайным образом (устарела; используйте `shuffle`) |
| `lower_bound(first, last, value)` | Возвращает итератор на первый элемент **≥ value** (требует сортировки) |
| `upper_bound(first, last, value)` | Возвращает итератор на первый элемент **> value** (требует сортировки) |
| `unique(first, last)`  | Удаляет последовательные дубликаты (возвращает итератор на новый конец) |

Примеры:

```cpp
vector<int> v = {2, 3, 3, 5, 7, 8, 8, 8};
sort(v.begin(), v.end());

// lower_bound и upper_bound
auto a = lower_bound(v.begin(), v.end(), 5);
auto b = upper_bound(v.begin(), v.end(), 5);
cout << *a << " " << *b; // 5 7

// Удаление дубликатов
v.erase(unique(v.begin(), v.end()), v.end());
// теперь v = {2,3,5,7,8}
```

## Полный пример программы

Вот программа, демонстрирующая объявление векторов, основные операции, доступ к элементам и работу с итераторами:

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    // 1. Создание и инициализация
    vector<int> vec1;                         // пустой
    vec1 = {0, 1, 2, 3};                      // присваивание списка

    vector<int> vec2 = {1, 2, 3, 4};          // инициализация

    vector<int> vec3(5, 0);                   // 5 нулей

    vector<vector<int>> matrix = {            // двумерный вектор
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

    vec1 = vec2;                              // глубокое копирование

    // 2. Основные операции
    vec1.push_back(10);                       // {1,2,3,4,10}
    vec1.pop_back();                          // {1,2,3,4}
    vec2.clear();                             // vec2 пуст, vec1 не изменился

    // 3. Доступ к элементам
    cout << "vec1[0] = " << vec1[0] << endl;
    cout << "vec1.front() = " << vec1.front() << endl;
    cout << "vec1.back() = " << vec1.back() << endl;
    cout << "*vec1.data() = " << *vec1.data() << endl;

    // 4. Заполнение в цикле
    for (int i = 0; i < 11; ++i) {
        vec1.push_back(i);
        cout << vec1.back() << " "; // выводим только что добавленное
    }
    cout << endl;

    // 5. Итераторы и алгоритмы
    sort(vec1.begin(), vec1.end());
    reverse(vec1.begin(), vec1.end());
    // random_shuffle устарел, лучше использовать shuffle:
    // shuffle(vec1.begin(), vec1.end(), default_random_engine{});

    // 6. Вывод финального вектора
    cout << "Final vector (" << vec1.size() << " elements): ";
    for (int x : vec1) {
        cout << x << " ";
    }
    cout << endl;

    return 0;
}
```

## Двумерные векторы (матрицы)

Вектор векторов позволяет создавать динамические матрицы:

```cpp
vector<vector<int>> matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

cout << matrix[1][2]; // 6 (строка 1, столбец 2)
```

Доступ к элементам матрицы осуществляется через два индекса: первый — номер строки, второй — номер столбца.

## Важные замечания

- Вектор автоматически управляет памятью: при добавлении новых элементов он перераспределяет внутренний буфер, если места недостаточно.
- Итераторы недействительны после вставки/удаления элементов, которые могут вызвать реаллокацию.
- Для эффективной работы с большими объёмами данных можно использовать `reserve()` для предварительного выделения памяти.

```cpp
vector<int> v;
v.reserve(1000); // выделяем память под 1000 элементов заранее
```
