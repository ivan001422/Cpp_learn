# 📘 Множества (`std::set` и `std::multiset`)

Множество — это контейнер, который хранит уникальные элементы в отсортированном порядке. Основные операции (вставка, удаление, поиск) выполняются за логарифмическое время. В C++ доступны две основные разновидности: **упорядоченные** (`std::set`, `std::multiset`) на основе сбалансированного двоичного дерева (красно-чёрное дерево) и **неупорядоченные** (`std::unordered_set`, `std::unordered_multiset`) на основе хеш-таблиц. Здесь мы сосредоточимся на упорядоченных версиях, так как они предоставляют дополнительные возможности (нижняя/верхняя граница, обход в порядке возрастания).

---

## Подключение и объявление

Для использования множеств подключите заголовочный файл:

```cpp
#include <set>      // для std::set и std::multiset
```

Объявление:

```cpp
std::set<int> s;                 // пустое множество целых чисел
std::set<int, std::greater<int>> s_desc; // множество в порядке убывания
std::set<int> s2 = {3, 1, 4, 1, 5}; // инициализация списком (дубликаты игнорируются)
```

Для мультимножества:

```cpp
std::multiset<int> ms;           // может содержать повторяющиеся элементы
```

## Основные методы (set)

| Метод                      | Описание                                                                 | Сложность |
|----------------------------|--------------------------------------------------------------------------|-----------|
| `insert(value)`            | Вставляет элемент. Для `set` – если уже есть, ничего не делает.          | O(log n)  |
| `erase(value)`             | Удаляет все элементы с данным значением (для `set` – не более одного).   | O(log n)  |
| `erase(iterator)`          | Удаляет элемент по итератору.                                            | O(1) амортизировано |
| `find(value)`              | Возвращает итератор на элемент или `end()`, если не найден.              | O(log n)  |
| `count(value)`             | Возвращает 1 (если есть) или 0 (если нет) для `set`; для `multiset` – количество. | O(log n) |
| `lower_bound(value)`       | Итератор на первый элемент **не меньше** value.                          | O(log n)  |
| `upper_bound(value)`       | Итератор на первый элемент **больше** value.                             | O(log n)  |
| `begin()`, `end()`         | Итераторы для обхода (элементы отсортированы по возрастанию).            | O(1)      |
| `rbegin()`, `rend()`       | Обратные итераторы для обхода в обратном порядке.                        | O(1)      |
| `size()`                   | Количество элементов.                                                   | O(1)      |
| `empty()`                  | Проверка на пустоту.                                                     | O(1)      |
| `clear()`                  | Удаляет все элементы.                                                    | O(n)      |

## Пример использования set

```cpp
#include <iostream>
#include <set>

int main() {
    std::set<int> s;
    s.insert(3);
    s.insert(2);
    s.insert(5);
    s.insert(2); // игнорируется, так как 2 уже есть

    std::cout << "Размер: " << s.size() << '\n'; // 3

    // Поиск
    if (s.find(3) != s.end())
        std::cout << "3 найдено\n";

    std::cout << "count(2): " << s.count(2) << '\n'; // 1
    std::cout << "count(4): " << s.count(4) << '\n'; // 0

    // Обход в порядке возрастания
    std::cout << "Элементы множества: ";
    for (int x : s)
        std::cout << x << " "; // 2 3 5
    std::cout << '\n';

    // lower_bound и upper_bound
    auto it_low = s.lower_bound(3); // указывает на 3
    auto it_up = s.upper_bound(3);  // указывает на 5
    std::cout << "*lower_bound(3) = " << *it_low << '\n';
    std::cout << "*upper_bound(3) = " << *it_up << '\n';

    // Удаление
    s.erase(3);
    std::cout << "После удаления 3: ";
    for (int x : s) std::cout << x << " "; // 2 5
    std::cout << '\n';

    return 0;
}
```

## Мультимножество (`std::multiset`)

В мультимножестве можно хранить несколько одинаковых элементов. Все методы аналогичны `set`, но поведение некоторых отличается:

- `insert(value)` всегда добавляет новый элемент (даже если уже есть такие же).
- `count(value)` возвращает количество элементов с данным значением.
- `erase(value)` удаляет **все** копии значения.
- `erase(iterator)` удаляет только один элемент (по указанной позиции).

### Пример использования multiset

```cpp
#include <iostream>
#include <set>

int main() {
    std::multiset<int> ms;
    ms.insert(5);
    ms.insert(5);
    ms.insert(5);
    ms.insert(2);
    ms.insert(3);

    std::cout << "Количество 5: " << ms.count(5) << '\n'; // 3

    // Обход
    std::cout << "Все элементы: ";
    for (int x : ms) std::cout << x << " "; // 2 3 5 5 5
    std::cout << '\n';

    // Удаление всех 5
    ms.erase(5);
    std::cout << "После erase(5): ";
    for (int x : ms) std::cout << x << " "; // 2 3
    std::cout << '\n';

    // Удаление только одной копии
    ms.insert(7);
    ms.insert(7);
    auto it = ms.find(7); // находим первую копию
    ms.erase(it);         // удаляем одну копию
    std::cout << "После удаления одной 7: ";
    for (int x : ms) std::cout << x << " "; // 2 3 7
    std::cout << '\n';

    return 0;
}
```

### Важное замечание по производительности

- `count(value)` в мультимножестве работает за O(log n + k), где k — количество найденных элементов. Если нужно просто проверить наличие, лучше использовать `find()`.
- `erase(value)` удаляет все копии, что может быть дорого, если копий много. Если нужно удалить одну, используйте `erase(find(value))`.

## Упорядоченность и итераторы

Элементы в `std::set` и `std::multiset` всегда хранятся в отсортированном порядке (по умолчанию по возрастанию). Это позволяет:

- Использовать алгоритмы, требующие отсортированного диапазона (например, `std::set_intersection`).
- Эффективно искать минимальный и максимальный элементы через `*s.begin()` и `*s.rbegin()`.
- Применять `lower_bound` и `upper_bound` для поиска диапазона значений.

Итераторы множеств являются **двунаправленными** (можно двигаться `++` и `--`), но не произвольного доступа (нельзя `it + n`).

## Выбор между set и unordered_set

| Критерий              | `set`                              | `unordered_set`               |
|-----------------------|------------------------------------|-------------------------------|
| Упорядоченность       | Да (по ключу)                      | Нет (случайный порядок)       |
| Сложность операций    | O(log n)                           | O(1) в среднем, O(n) в худшем |
| Дополнительные методы | `lower_bound`, `upper_bound`       | Отсутствуют                   |
| Память                | Меньше накладных расходов          | Больше из-за хеш-таблицы      |
| Использование         | Когда важен порядок или нужны диапазонные операции | Когда нужна максимальная скорость и порядок не важен |

---

## Полезные алгоритмы с множествами

Множества часто используются вместе с алгоритмами из `<algorithm>`:

- `std::set_intersection` – пересечение двух множеств.
- `std::set_union` – объединение.
- `std::set_difference` – разность.
- `std::includes` – проверка, является ли одно множество подмножеством другого.

Все эти алгоритмы требуют отсортированных диапазонов, что идеально подходит для `set`.
