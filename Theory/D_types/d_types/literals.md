# Литералы в C++

**Литералы (константы)** — это фиксированные значения, которые записываются непосредственно в коде и не могут быть изменены программой.  
Каждый литерал имеет определённый тип, зависящий от его записи и суффиксов.

## Символьные литералы

Заключаются в одинарные кавычки.  
Префиксы задают тип символа (C++11 и новее):
```
| Запись     | Тип        | Описание                |
|------------|------------|-------------------------|
| `'A'`      | `char`     | обычный символ          |
| `u'A'`     | `char16_t` | UTF-16                  |
| `U'A'`     | `char32_t` | UTF-32                  |
| `L'A'`     | `wchar_t`  | широкий символ          |
```
## Целочисленные литералы

Записываются как числа без дробной части.  
Суффиксы уточняют тип (регистр не важен, но `l` лучше писать заглавной):
```
| Литерал     | Тип                      |
|-------------|--------------------------|
| `123`       | `int` (по умолчанию)     |
| `123u`      | `unsigned int`           |
| `123l`      | `long`                   |
| `123ll`     | `long long` (C++11)      |
| `123ul`     | `unsigned long`          |
| `123ull`    | `unsigned long long` (C++11) |
```
## Литералы с плавающей точкой

Содержат десятичную точку или экспоненту.  
По умолчанию имеют тип **`double`**.  
Суффиксы меняют точность:
```
| Литерал   | Тип           |
|-----------|---------------|
| `3.14`    | `double`      |
| `3.14f`   | `float`       |
| `3.14l`   | `long double` |
```
## Правила определения типа компилятором

- **Целые числа**: выбирается наименьший подходящий тип, начиная с `int`.  
  (Пример: в 16-битной среде `10` → `int`, `103000` → `long int`).  
  Исключение: значение не будет усечено до `char`, если оно в него не помещается.
- **Плавающая точка**: по умолчанию используется `double`.

Явные суффиксы переопределяют эти правила и гарантируют нужный тип.

## Шестнадцатеричные и восьмеричные литералы

- **Восьмеричные**: префикс `0` (ноль). Пример: `011` = 9 в десятичной.
- **Шестнадцатеричные**: префикс `0x` или `0X`. Пример: `0xFF` = 255.

```cpp
int hex = 0xFF;   // 255
int oct = 011;    // 9
```

## Управляющие последовательности

Вставляются в символьные и строковые литералы для представления специальных символов.  
Начинаются с обратного слеша `\`.

| Код   | Значение                              |
|-------|---------------------------------------|
| `\b`  | Возврат на одну позицию (Backspace)   |
| `\f`  | Подача страницы (Form feed)           |
| `\n`  | Новая строка (New line)               |
| `\r`  | Возврат каретки (Carriage return)     |
| `\t`  | Горизонтальный отступ (Tab)           |
| `\"`  | Двойная кавычка                       |
| `\'`  | Одинарная кавычка (апостроф)          |
| `\\`  | Обратный слеш                         |
| `\v`  | Вертикальный отступ (Vertical tab)    |
| `\a`  | Звуковой сигнал (Alert/Bell)          |
| `\?`  | Вопросительный знак (trigraph)        |
| `\N`  | Восьмеричный код (N – цифры 0-7)      |
| `\xN` | Шестнадцатеричный код (N – hex цифры) |

### Пример использования

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Привет,\nмир!\n";             // Перевод строки
    cout << "Табуляция:\tтекст\n";         // Горизонтальный отступ
    cout << "Кавычки: \"C++\"\n";          // Двойные кавычки
    cout << "Звук:\a\n";                   // Звуковой сигнал
    cout << "Обратный слеш: \\n\n";        // Вывод \n как текста
    cout << "Восьмеричный код A: \101\n";  // 'A' (ASCII 65 восьм.)
    cout << "Шестнадцатеричный код A: \x41\n"; // 'A' (ASCII 65 hex)
    return 0;
}
```

> **Примечание:** управляющая последовательность интерпретируется компилятором как единый символ, а не как несколько обычных знаков.
