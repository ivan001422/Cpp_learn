## Аргументы командной строки

Функция `main()` может принимать два необязательных параметра для обработки аргументов, переданных программе при запуске.

```cpp
#include <iostream>
using namespace std;

int main(int argc, char* argv[]) {
    // ...
}
```

| Параметр | Назначение |
|----------|------------|
| `argc`   | **Argument count** – количество переданных аргументов (всегда ≥ 1) |
| `argv`   | **Argument vector** – массив C-строк, содержащих сами аргументы |

Имена `argc` и `argv` приняты по соглашению, но формально могут быть любыми.

### Структура argv

- `argv[0]` – имя запущенной программы (или пустая строка, если имя недоступно).
- `argv[1]` ... `argv[argc-1]` – аргументы, разделённые пробелами или табуляцией.
- `argv[argc]` гарантированно равен `nullptr`.

Все аргументы передаются **как строки**, поэтому числовые данные нужно преобразовывать явно.

### Пример: вывод всех аргументов

```cpp
#include <iostream>
using namespace std;

int main(int argc, char* argv[]) {
    cout << "Количество аргументов: " << argc << '\n';
    for (int i = 0; i < argc; ++i) {
        cout << "argv[" << i << "] = " << argv[i] << '\n';
    }
    return 0;
}
```

### Передача строк с пробелами

При запуске из командной строки аргументы разделяются пробелами. Чтобы передать один аргумент, содержащий пробел, его нужно **заключить в кавычки**:

```bash
./program "hello world" 42
```

### Преобразование числовых аргументов

Библиотека `<cstdlib>` предоставляет функции для преобразования строк в числа:

| Функция   | Возвращаемый тип |
|-----------|------------------|
| `atoi()`  | `int`            |
| `atol()`  | `long int`       |
| `atoll()` | `long long`      |
| `atof()`  | `double`         |

Пример:

```cpp
#include <iostream>
#include <cstdlib>
using namespace std;

int main(int argc, char* argv[]) {
    if (argc < 3) {
        cerr << "Использование: " << argv[0] << " <число1> <число2>\n";
        return 1;
    }
    int a = atoi(argv[1]);
    int b = atoi(argv[2]);
    cout << a << " + " << b << " = " << a + b << '\n';
    return 0;
}
```

> **Примечание:** функции `atoi`/`atof` не сообщают об ошибках преобразования (некорректная строка даст 0). Для безопасного разбора используйте `stoi`, `stod` из `<string>` (также доступны с `using namespace std;`).

# Указатели и ссылки в C++

**Указатель** хранит адрес переменной.  
**Ссылка** — это псевдоним уже существующей переменной.

## Ключевые различия

| Указатель (`*`)                         | Ссылка (`&`)                           |
|-----------------------------------------|----------------------------------------|
| Может быть объявлен без инициализации   | Должна быть инициализирована сразу     |
| Можно перенаправить на другой объект    | Всегда указывает на один и тот же объект|
| Может быть `nullptr`                    | Не может быть «пустой»                 |
| Доступ к значению через `*p`            | Используется как обычная переменная     |
| Занимает память (размер адреса)         | Обычно не занимает дополнительной памяти|

## Пример

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[3] = { 1, 2, 3 };

    int *p = arr;        // p хранит адрес arr[0]
    int &l = arr[0];     // l — псевдоним arr[0]

    cout << "p = " << p << ", l = " << l << '\n';  // адрес и 1

    p = &arr[1];         // теперь p указывает на arr[1]
    l = arr[1];          // присваивание: arr[0] = 2 (ссылка не перепривязывается)

    cout << "p = " << p << ", l = " << l << '\n';  // адрес arr[1] и 2
    cout << "arr[0] = " << arr[0] << ", arr[1] = " << arr[1] << '\n'; // 2 и 2

    *p = 100;            // arr[1] = 100
    cout << "arr[1] = " << arr[1] << '\n';  // 100

    return 0;
}
```

**Вывод:**
```
p = 0x... , l = 1
p = 0x... , l = 2
arr[0] = 2, arr[1] = 2
arr[1] = 100
```

> Ссылка даёт новое имя объекту; все действия с ней — это действия с исходной переменной. Указатель же можно переназначать на другие объекты и проверять на `nullptr`.
