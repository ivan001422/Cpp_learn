# Инкапсуляция

Инкапсуляция — это механизм программирования, связывающий воедино код и обрабатываемые им данные,
чтобы защитить их от внешнего вмедиательства и неправильного использования. В объектно-ориентированном 
языке код и данные могут быть связаны путем создания независимого "черного ящика". Он содержит все
необходимые (для обеспечения самостоятельности) данные и код. При таком связывании создается объект — 
это конструкция, поддерживающая инкапсуляцию.

Базовой единицей инкапсуляции в С++ называется "классом". Он определяет новый тип данных, который задает
формат объекта. Класс содержит как данные, так и код, предназначенный для обработки этих данных.
Следовательно, класс связывает данные с кодом. В С++ спецификация класса используется для построения объектов,
то есть экземпляров класса. По сути, класс представляет собой набор планов, которые определяют, как строить объект.

В классе данные объявляются в виде переменных, а код оформляется в виде функций (подпрограмм). Функции и
переменные, составляющие класс, называются его членами. Таким образом, переменная, объявленная в классе, —
это член данных (также используется понятие "переменная экземпляра"), а функция, объявленная в классе, —
функция-член.
