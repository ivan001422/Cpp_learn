1. Инструкция (Statement)

Инструкция: минимальная законченная единица программы, которую компилятор может выполнить. Программа — это последовательность инструкций.

Примеры:

```
int x = 5;          // инструкция объявления с инициализацией
x = x + 1;          // инструкция выражения (вычисление и присваивание)
return 0;           // инструкция возврата
;                   // пустая инструкция (ничего не делает)
```

2. Оператор (Operator)

Оператор: Символ или ключевое слово, которое выполняет операцию над одним или несколькими операндами (данными).

   Категория               Примеры                 Что делают
Арифметические       |    +, -, *, /, %     |Сложение, вычитание и т.д.
Сравнения            | ==, !=, <, >, <=, >= |Сравнивают значения, возвращают bool
Логические           |        &&, ||        |Логическое сложение, умножение
Присваивания         |   =, +=, -=, *=, /=  |Присваивают и модифицируют
Инкремента/декремента|       ++, –-         |Увеличивают/уменьшают на 1

Примеры:

```
int a = 5 + 3;      // оператор '+' (сложение)
bool b = (a == 8);  // оператор '==' (сравнение)
a += 2;             // оператор '+=' (присваивание с добавлением)
a++;                // оператор '++' (инкремент)
```

3. Условные операторы (Conditional Statements)

Условные операторы: инструкции, которые позволяют выполнять код в зависимости от условия.

Название          Синтаксис                              Пример
if      |     if (условие) { ... }            | Выполнить, если условие истинно
else    |     else { ... }                    | Выполнить, если условие ложно
else if |     else if (условие) { ... }       | Проверить новое условие, если предыдущее ложно
switch  |     switch (выражение) { case ... } | Множественный выбор по значению

Примеры:

```
int score = 85;

if (score >= 90) cout « "Отлично!";


else if (score >= 75) cout « "Хорошо!";  // Выполнится это


else cout « "Нужно стараться";

char grade = 'B';
switch (grade) {
    case 'A': cout « "5"; break;
    case 'B': cout « "4"; break;  // Выполнится это
    default: cout « "3";
}
```

4. Литералы (Literals)

Литералы: константные значения, записанные прямо в коде.

Тип литерала                        Примеры
Целочисленные      | 42, -10, 0xFF (шестнадцатеричный), 075 (восьмеричный)
С плавающей точкой | 3.14, 2.5f, 1.0e-5 (научная нотация)
Символьные         | 'A', '\n' (перенос строки), '\x41' (ASCII код)
Строковые          | "Hello", "C:\files\" (экранирование обратного слэша)
Логические         | true, false
Указатель          | nullptr

Примеры:

```
int age = 25;               // 25 - целочисленный литерал
double pi = 3.14159;        // 3.14159 - литерал double
char letter = 'Q';          // 'Q' - символьный литерал
string name = "Alice";      // "Alice" - строковый литерал
bool isReady = true;        // true - логический литерал
```

5. Модификаторы (Modifiers)

Модификаторы: ключевые слова, изменяющие свойства базовых типов данных.

Модификатор         Применяется к                Что делает
signed       | char, int(стоит по умолчанию)  | Знаковый тип
unsigned     | char, int                      | Беззнаковый тип (только ≥0)
short        | int                            | Уменьшает диапазон значений (обычно 2 байта)
long         | int, double                    | Увеличивает диапазон/точность
const        | Любой тип                      | Значение нельзя изменить после инициализации
volatile     | Любой тип                      | Значение может измениться извне (оборудование, поток)

Примеры:

```
unsigned int count = 100;   // count всегда ≥ 0
short temperature = -5;     // экономит память
long bigNumber = 1000000L;  // больше обычного int

const double GRAVITY = 9.81; // константа, изменить нельзя
// GRAVITY = 10.0; // ОШИБКА компиляции!

volatile int sensorValue; // компилятор не оптимизирует доступ
```

6. Типы данных (Data Types)

Типы данных: определяет, какие данные может хранить переменная и какие операции с ней возможны.

Категория                  Примеры                               Описание
Целочисленные      |  int, short, long, char             |  Хранят целые числа
С плавающей точкой |  float, double, long double          |  Дробные числа
Логический         |  bool                                |  true или false
Символьный         |  char, wchar_t                       |  Символы ASCII/Unicode
Производные        |  массивы, указатели, ссылки, функции | Зависит от базовых

Примеры:

```
int apples = 10;
float price = 99.99f;
bool isOpen = false;
char symbol = '$';
int numbers[5] = {1, 2, 3, 4, 5}; // массив
```

7. Переменные (Variables)

Переменные: именованная область памяти для хранения данных.

Примеры:

```
int score;          // объявление переменной
score = 100;        // инициализация (присваивание значения)
int height = 180;   // объявление с инициализацией
```

8. Функции (Functions)

Функция: блок кода, который выполняет конкретную задачу и может быть вызван по имени.

Примеры:

```
// Объявление функции
int add(int a, int b);

// Определение функции
int add(int a, int b) {
    return a + b;  // возвращаемое значение
}

// Вызов функции
int result = add(5, 3); // result = 8
```
